#MongoDB相关资料

1、安装MongoDB

	* python3安装pymongo模块
		pip3 install pymongo
		pip3 list #查看


2、相关操作
	show databases;
	use sites;
	show collections;
	db.sites.find().pretty()

3、连接MongoDB

	from pymongo import MongoClient
	conn = MongoClient('192.168.0.113', 27017)
	db = conn.mydb  #连接mydb数据库，没有则自动创建
	my_set = db.test_set　　#使用test_set集合，没有则自动创建
